# 📊 Employee Salary Analytics

## 📌 Project Overview

This project analyzes employee salary data using Python and Pandas. It demonstrates how data analytics can help Human Resources (HR) make informed decisions regarding salaries, promotions, hiring, and departmental performance.

This is **Project 02** of my **Mission Alpha – Data Analytics Portfolio**.

---

## 🎯 Objectives

- Analyze employee salary information.
- Identify highest and lowest paid employees.
- Filter employees based on salary, department, and performance.
- Sort employee data by salary and experience.
- Perform department-wise analysis using Pandas.
- Generate business insights for HR decision-making.

## 📈 Analysis Performed

### Basic Analysis

- Total number of employees
- Average salary
- Highest salary employee
- Lowest salary employee

### Filtering

- Employees earning more than ₹50,000
- Employees with Excellent performance
- Mechanical department employees

### Sorting

- Employees sorted by salary
- Employees sorted by experience

### Group Analysis

- Average salary by department
- Employee count by department

### Bonus Analysis

- Employee with highest Salary per Year of Experience

## 💼 Business Insights

The project helps answer important HR questions such as:

- Which employee should be promoted?
- Which department needs more hiring?
